## My process
I started by structuring the HTML with semantic elements, creating a main container for the QR code card, the image, and the text. Then I used CSS to style the card, centering it on the page with Flexbox, adding padding, borders, and text styles to make it visually appealing. The project was straightforward, focusing on practicing layout, spacing, and basic styling with HTML and CSS.

### Built with

- Semantic HTML5 markup
- CSS custom properties
- Flexbox

### What I learned
This project helped me practice structuring a simple webpage with semantic HTML and styling it with CSS. I focused on using Flexbox to center elements and create a clean card layout. I also practiced adding images and styling text for a visually appealing component.
